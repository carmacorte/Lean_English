# Modal Verbs

Modal verbs are auxiliary verbs used to express obligation, advice, possibility, future intention, and hypothetical situations.

In Spanish, many modal verbs are translated as forms of **deber**, **tener que**, **debería**, **haré**, or **haría**, depending on the context.

## Topics

| Modal Verb | Main Use | Example | Spanish Meaning |
|---|---|---|---|
| Must | Strong obligation / logical deduction | You must wear a seatbelt. | Debes usar cinturón de seguridad. |
| Have to | External obligation / necessity | I have to finish my homework. | Tengo que terminar mi tarea. |
| Should | Advice / recommendation | You should eat more vegetables. | Deberías comer más verduras. |
| Shall | Offer / suggestion | Shall we go to the park? | ¿Vamos al parque? |
| Will | Future / promise | I will call you later. | Te llamaré más tarde. |
| Would | Conditional / polite request | Would you like some coffee? | ¿Te gustaría café? |

## Learning Path

1. Understand the function of each modal verb.
2. Learn the sentence structure.
3. Compare English examples with Spanish meaning.
4. Practice with short translations.
5. Apply the modal verbs in real-life situations.

## General Sentence Pattern

```text
Subject + modal verb + base verb
```

Examples:

```text
You must wear a seatbelt.
I should study more.
She will call later.
Would you like some coffee?
```

## Important Rule

After a modal verb, the next verb usually stays in its base form.

Correct:

```text
You should eat more vegetables.
```

Incorrect:

```text
You should eats more vegetables.
```
